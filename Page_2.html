Voyons voir si on peut faire un bon usage des @keyframes.

Pour ce faire, on utilise d'abord un petit filtre pour voir ce que ça donne, le hue-rotate() ajouté dans le CSS et affecté à ma classe ou mon id. Quand on a notre idée (ça peut être n'importe quel paramêtre du CSS qu'on veut faire varier dans le temps), on va se renseigner sur les @keyframes. Comme on peut le constater, l'élément CSS que je veut influencer a une ligne "animation-name: colorChanging", puis d'autres. Quand vous ferez votre @keyframe, c'est ce nom que vous utiliserez pour viser l'élément.
